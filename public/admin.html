Access Control

Manage who can upload, view, and manage files.

Folder Access

Assign a top-level folder to an access list. Members of the list get the chosen level in that folder tree; everyone else gets none (admins excepted). The list default means everyone — including anonymous visitors — which is how a folder is deliberately published. Unassigned folders follow the agent access control above.
